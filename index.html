Education

Ph.D. — Computer Science, The Hong Kong University of Science & Technology, Sep 2009–Jun 2013

Thesis Title: Efficient Processing of Complex Join Queries on the Cloud.

Investigating the data allocation, computation scheduling, and optimal network shuffle to answer multi-way join type queries, a.k.a "complex join" queries efficiently. Particularly, given a number of processing units, we map a complex join query to a number of parallel tasks and having them executed in a well-scheduled sequence, such that the total processing time span is minimized. In this thesis, we demonstrate how our “complex join” solution can be well applied to the query processing over various data analytic scenarios, i.e., querying RDF data, pattern matching over graph data and etc.

During my candidature I was, and continue to be, an active member of the database community, having attended and presented at multiple conferences and workshops. Including SIGMOD, VLDB, and ICDE.

 

MSc with Distinction — Computer Systems & Architecture, University of Science & Technology Beijing, Sep 2006–Jan 2009

Thesis Title: The New User Problem in Collaborative Filtering.

The new user problem is a problem that all collaborative filtering systems must face. How can the system make recommendations for a user when it does not know what that user likes? I developed new methods for a collaborative filtering system to choose which items to present to a new user for them to rate. This technique of presenting items to the user forms a method by which the new-user problem can be alleviated for collaborative filtering systems. Developed new metrics for evaluation of, and comparison between, the different methods a system might choose to select the next item.

For my final year research project, I worked on a cluster computing management toolkit, which performs real-time system monitoring, job scheduling, data migration and automatic backup. My focus is the job scheduling module that promises as much total throughput as possible.
